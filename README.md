# caderno-financeiro-notebooklm
Este projeto mostra como a IA, via NotebookLM, pode apoiar a aprendizagem ativa em educação financeira. Com curadoria de fontes, prompts e pensamento crítico, o conteúdo virou um miniguia com resumos, glossário e prompts reutilizáveis. O foco está no resultado e no processo de construção do conhecimento.
